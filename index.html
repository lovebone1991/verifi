What Verifi checks

Layer 1 · Structural

Is the model well-formed?

  • Formula errors (#REF, #DIV/0, #NAME)
  • Hardcoded values in calculation cells
  • Circular references
  • Inputs sheet separation
  • Centralised toggle controls
  • Orphaned input cells
Layer 2 · Accounting

Does the maths close?

  • Cash flow identity (period by period)
  • Debt drawdown = repayment
  • Sources = Uses
  • Interest fully repaid
  • Levered CF = Unlevered + debt schedule
  • Actual → forecast transition
Layer 3 · Economic

Do the numbers make sense?

  • Yield on cost vs completion cap rate
  • Leverage uplift vs LTV / LTC
  • Return waterfall reconciliation
  • Income return vs ingoing yield
  • Capital growth vs rental growth
  • Development margin benchmarks

Privacy & data
